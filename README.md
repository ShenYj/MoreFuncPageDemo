# MoreFuncPageDemo

更多定制功能页demo, 这是一个Swift 版本

- [x] 已选、可选展示  
- [x] 已选区域编辑顺序  
- [x] 可选区域分组为空时隐藏该组  
- [x] 已选区组长按进入编辑状态
